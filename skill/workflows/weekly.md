# 周报流程

## 流程概览

```
0: 全局同步 → 1: 生成周报
```

---

## 第0步：全局同步

**工具**：`system_sync()` 【MCP工具】

**目的**：同步所有私聊的最新消息。

**执行细节**：
- 全局同步可能耗时较长（取决于消息量）
- 如果只需要排名快照，可以用 `system_sync(meta_only=True)`

---

## 第1步：生成周报

**工具**：`weekly_report()` 【MCP工具】

**目的**：生成本周客户维护总结。

**执行细节**：
- 返回排名快照 + 趋势分析 + 维持建议
- 包含需要关注的客户和需要跟进的商机
