技术架构

SalesCRM 采用四层架构设计：

  • 数据层：SQLite + Markdown 事实档案
  • 服务层：指标计算、排名引擎、事件检测
  • 工具层：统一 Agent API，16+ 工具函数
  • 客户端：Web 仪表盘 + AI Agent

核心原则：代码负责数据，Agent 负责推理。所有量化指标由 Python 计算，AI 专注于分析和决策建议。
