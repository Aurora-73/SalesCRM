## 项目概述

《北狩》是一款基于**严格史实**的历史沉浸式决策游戏。玩家将穿越回1860年第二次鸦片战争的关键时刻，化身咸丰皇帝，在帝国存亡之际做出艰难抉择。

### 核心创新点

1. **史实即玩法**：所有游戏内容均依据《清史稿》《筹办夷务始末》等正史，拒绝"戏说历史"
2. **AI旁白系统**："历史之声"全程伴随，在你偏离史实时温柔纠正，在你犹豫时蛊惑引导
3. **多维度决策**：每个选择都有史实评判，没有对错，只有"是否符合历史"
4. **假设推演引擎**：当你做出与史实不同的选择时，游戏会基于史料进行合理推演
5. **跨平台即开即玩**：Web端 + 移动端原生兼容，无需下载安装

## 核心玩法

### 1. 自由探索
- 以第一人称视角在紫禁城内行走
- 与大臣对话收集情报
- 发现隐藏的历史细节

### 2. NPC对话系统
每位NPC都有基于史实的对话树：
- **恭亲王奕訢**：留守派，提供防务与和谈情报
- **僧格林沁**：主战派，诉说八里桥惨败
- **肃顺**：近臣，劝说北狩
- **懿贵妃**：后宫视角，担忧皇子与皇帝身体
- **载淳**：6岁皇子，童言无忌

### 3. 关键决策点
游戏设置4个核心决策点，均取材于真实历史：

| 决策点 | 历史背景 | 选项数 |
|--------|---------|--------|
| 北狩前夜 | 八里桥惨败后是否召集重臣 | 4 |
| 人事安排 | 任命谁留守北京 | 3 |
| 最后决断 | 是否批准僧王再战 | 3 |
| 热河忧思 | 收到圆明园被占消息后的应对 | 3 |

### 4. AI旁白系统
- **场景旁白**：进入新场景时的氛围描述
- **闲置旁白**：玩家无操作时触发历史氛围台词
- **决策评论**：每个选择后给出"历史之声"的评论
- **史实提示**：适时弹出经过验证的历史知识

### 5. 结局系统
- **历史轨迹**：做出与史实一致的选择，呈现真实历史结局
- **假设推演**：做出不同选择，基于史料进行合理推演

## 史实准确性保障

### 史料来源
1. **《清史稿》**：清代官方正史
2. **《筹办夷务始末·咸丰朝》**：清代外交档案汇编
3. **学术论文与专著**：二次鸦片战争相关研究

### 核查机制
- 所有NPC对话均标注史料出处
- 每个决策选项都有"是否符合史实"的明确标记
- 游戏内置"史实记录"面板，记录玩家所有选择与史实对照
- 严禁出现的内容清单：
  - [x] 咸丰与慈禧的"宫斗"（此时尚未发生）
  - [x] 八国联军（发生于1900年）
  - [x] 太平天国与第二次鸦片战争的混淆
  - [x] 超出时代的技术

## 运行方式

### 本地运行
```bash
# 方式1：直接打开
双击 index.html 即可在浏览器中运行

# 方式2：简易HTTP服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 部署上线
由于所有资源均为本地文件（无外部API依赖），可直接部署到：
- GitHub Pages
- Vercel / Netlify
- 任何静态文件托管服务

### 移动端适配
- 自动检测触摸设备，显示虚拟摇杆
- UI自适应不同屏幕尺寸
- 禁止页面滚动，确保游戏体验
