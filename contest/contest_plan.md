## 五、创意提案内容要点

### 1. 创意名称 + 创意介绍
- 想解决什么问题：销售缺乏系统化的客户分析工具，无法准确判断客户意向和跟进时机
- 为什么会想到做这个：基于智能分析系统的架构，专门为销售场景设计，帮助销售提升转化率
- 大概是什么产品：AI Agent + Web 管理平台

### 2. 目标用户及痛点
- 面向哪些用户：B2B/B2C 销售、客户经理、销售团队管理者
- 在什么场景下使用：日常客户跟进、周报分析、商机评估、客户维护
- 当前痛点：依赖人工经验判断，缺乏数据量化；客户信息分散，无法统一分析；跟进时机凭感觉

### 3. 价值与意义
- 效率提升：自动化分析客户状态，减少人工判断时间
- 商业价值：提升成交率和客户转化率

### 4. 核心功能亮点
- 16 个加权量化指标，科学评估客户状态
- 5 个销售决策公式（BQ/BSP/BWS/PV/BS），智能判断跟进策略
- AI Agent 自动分析，提供个性化建议
- 多数据源支持（微信/企微/CRM/邮件）
- 身份目录三层映射，支持多账号统一管理

## 六、技术架构亮点

```
用户 → AI Agent → Python 工具 → 数据层
                              ↓
                        知识库 + 决策公式
```

- **代码负责数据，Agent 负责推理**：核心原则
- **增量同步**：从微信/企微实时同步聊天记录
- **指标引擎**：自动计算 16 个销售指标
- **事件检测**：自动识别销售关键事件
- **知识库**：结构化销售知识支持
